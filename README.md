# Simple Nvim configuration for Python and Rust

## Prerequisites

- Neovim 0.12+
- Python 3.*
- Rust toolchains 
- ripgrep
- tree-sitter-cli
- Nerd font (recommended)

## Installation

### 1. Backup your current config

```bash
$ mv ~/.config/nvim ~/.config/nvim.bkp
$ git clone https://github.com/smrenato/nvim.git ~/.config/nvim 

```
